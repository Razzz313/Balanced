# Balance Logo Assets

## Files Included

- `logo-green.svg` — Full lockup (icon + wordmark) on white/light backgrounds
- `logo-white.svg` — Full lockup (icon + wordmark) on dark/green backgrounds  
- `logo-icon.svg` — Icon mark only (circle with fork/knife/plate)
- `favicon.svg` — Browser favicon (use in <link rel="icon" type="image/svg+xml">)

## Generating PNG Variants

To export PNG versions, open each SVG in:
- **Figma**: Import SVG → Export as PNG at 1x, 2x, 3x
- **Inkscape**: File → Export PNG (free, cross-platform)
- **Browser**: Open SVG → Right-click → Save as image

## Recommended Sizes

| Use Case | Size |
|----------|------|
| Website header | 200×60px |
| Favicon | 64×64px |
| App icon | 512×512px |
| Business card | 800×240px (300 DPI) |
| Social media | 400×400px (icon only) |
